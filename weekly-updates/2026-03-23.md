# Claude Weekly Update — March 23, 2026

> Covering the week of March 16–23, 2026

---

## Claude Code

### Shipped

- **Voice Mode** (`/voice`) — Push-to-talk via spacebar, rolling out progressively to all users. 10 new languages added (Russian, Polish, Turkish, Dutch, Ukrainian, Greek, Czech, Danish, Swedish, Norwegian).
- **`/loop` command** (v2.1.71) — Session-level cron job. Example: `/loop 5m check the deploy` polls every 5 minutes. Disable with `CLAUDE_CODE_DISABLE_CRON`.
- **`/effort` command** (v2.1.76) — Set model effort (low / medium / high). Use `ultrathink` keyword for a temporary max-effort burst.
- **`/color` command** (v2.1.75) — Color-code the prompt bar per session; handy for managing multiple parallel sessions.
- **1M token context window** — Available by default for Max, Team, and Enterprise plans (Opus 4.6, from v2.1.75).
- **Opus 4.6 as default model** — Default output tokens raised to 64k, upper bound to 128k.
- **MCP Elicitation** (v2.1.76) — MCP servers can now request structured input (forms, browser URLs) mid-task without interrupting the workflow.
- **`--channels` research preview** — MCP push messaging, improved permissions, autocomplete, and plugin install flows.
- **`--bare` flag** — For scripted `-p` calls; skips hooks, LSP, plugin sync, skill walks; requires `ANTHROPIC_API_KEY`.
- **Rate limits in statusline** — New `rate_limits` field exposes 5-hour and 7-day usage windows (`used_percentage`, `resets_at`).
- **`allowRead` sandbox setting** — Added alongside Opus 4.6 stability updates.

### Key Bug Fixes

- Multiple concurrent sessions no longer require repeated re-auth when one session refreshes its OAuth token.
- Fixed `--resume` dropping parallel tool results.
- Fixed auto-updater accumulating tens of GB of memory from overlapping binary downloads.
- Fixed Linux sandbox "ripgrep not found" error on native builds.
- Fixed native modules failing to load on Amazon Linux 2 / glibc 2.26.

### Version Range This Week

v2.1.63 → v2.1.76
