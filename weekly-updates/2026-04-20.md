# Claude Weekly Update — April 20, 2026

_Covering: April 13–20, 2026_

---

## Shipped This Week

### Claude Code Desktop Redesign (April 14)
Anthropic shipped a complete redesign of the Claude Code desktop app for Mac and Windows, built around parallel agent sessions. Key additions:
- **Multi-session sidebar** — manage several agent sessions side-by-side
- **Drag-and-drop pane layout** — arrange workspace however you like
- **Integrated terminal** — no more switching out of the app
- **In-app file editor** — review, spot-edit, and approve PRs without leaving Claude Code
- **Rebuilt diff viewer** — faster performance on large changesets
- **Expanded preview pane** — now handles HTML, PDFs, and local app servers

Available to Pro, Max, Team, and Enterprise users with Claude Code.

---

### Claude Code Routines (April 14 — Research Preview)
Routines let you define reusable agentic workflows that run without an active session. Each routine bundles a prompt, a repo, and connectors into a single config that can:
- Run on a schedule
- Fire from an API call
- Trigger off a GitHub event (e.g., new PR)

Available in research preview for Pro, Max, Team, and Enterprise users with Claude Code on the web enabled.

---

### Claude Opus 4.7 GA (April 16)
The latest Opus model shipped with meaningful upgrades over 4.6:
- **3× higher vision resolution**
- New **`xhigh` effort level** for more compute-intensive tasks
- **Task budgets** for controlling token spend per run
- Improved instruction following on complex, long-horizon coding tasks

---

### Claude Design Launch (April 17 — Research Preview)
A new Anthropic Labs product for creating visual outputs — designs, prototypes, slides, and one-pagers — in collaboration with Claude. Available as a research preview for Pro, Max, Team, and Enterprise subscribers.

---

### Claude Cowork — Now Generally Available
Claude Cowork reached GA on macOS and Windows via the Claude Desktop app this week. Pro and Max plan users get a persistent agent thread accessible from Desktop, iOS, and Android. New this week: scheduling both recurring and on-demand tasks directly in Cowork, plus a new **Customize** section that consolidates skills, plugins, and connectors in one place.

---

### Claude Code CLI Notable Changes
- **Ultraplan** (early preview) — draft a plan in the cloud from the CLI, edit it in a web UI, then run it remotely or pull it back local
- **Computer Use in CLI** (research preview) — Claude can open native apps, click through UI, and verify changes from your terminal
- **Monitor tool** — streams background events into the conversation so Claude can tail logs and react live
- `/loop` now self-paces when you omit the interval
- `/team-onboarding` packages your setup into a replayable guide
- `/autofix-pr` turns on PR auto-fix from your terminal
- **1-hour prompt cache TTL** opt-in added (API key, Bedrock, Vertex, Foundry)
- Session recap + improved resume, model switching, and memory use

---

### Claude Managed Agents (Public Beta)
A fully managed agent harness for running Claude autonomously: secure sandboxing, built-in tools, and server-sent event streaming. Create agents, configure containers, and run sessions through the API.

---

### Advisor Tool (Public Beta)
Pairs a fast executor model with a high-intelligence advisor model that provides strategic mid-generation guidance, delivering near-advisor-solo quality at executor-model token rates for long-horizon agentic workloads.

## Community Buzz

**Desktop redesign: love it and hate it.** The multi-session sidebar earned strong praise — "finally, no need to keep Macs running all night" was a common take. But token consumption concerns dominated the first 48 hours on X. Several developers reported burning through their full 5-hour usage quota in under 10 minutes on the new desktop before switching back to their previous setup.

**Routines are already in use.** Developers who got early access have shared their first Routine configs: PR review automation, issue triage, and post-deployment smoke testing are the three most common use cases showing up in the community.

**Cursor vs. Claude Code debate reignited.** The desktop redesign renewed comparisons with Cursor. A popular Substack post broke down how the new multi-session layout changes workflows relative to Cursor and other editors — with the conclusion that Claude Code is increasingly compelling as a standalone environment, not just a CLI.
