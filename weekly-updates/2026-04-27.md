# Claude Desktop & Claude Code Weekly Update
**Week of April 21–27, 2026**

---

## Shipped: New Features & Changes

### Claude Code Desktop Redesign (April 14)
Anthropic shipped a major redesign of the Claude Code experience inside the Claude desktop app:
- **Multi-session sidebar** — view, filter, and group all active and recent sessions by project or status
- **Integrated workspace** — built-in terminal, in-app file editor, rebuilt diff viewer for large changesets, and expanded preview pane (HTML, PDFs, local servers)
- **View modes** — Verbose, Normal, and Summary let you control how much tool-call activity is shown
- **Side chat shortcut** — `Cmd + ;` branches a question off a running task without polluting the main thread context

### Claude Code Routines (April 14)
New "Routines" feature lets you run automations without an active session. A routine bundles a prompt, a repo, and connectors into one config that can run on a schedule, fire from an API call, or trigger on GitHub events (e.g. new PR).

### Claude Cowork GA
Claude Cowork is now generally available on macOS and Windows via the Claude Desktop app.

### Claude Code CLI: Versions 2.1.69–2.1.117
A rapid release cadence this month shipped:
- **Native binary** — CLI moved from bundled JavaScript to a native binary (faster startup)
- **Persistent model selection** across sessions
- **Rewritten `/resume` command** and **inline thinking progress**
- **`/powerup` system** for interactive onboarding for new users
- **MCP 500K upgrade** — tool output character limit raised to 500,000
- **`/effort` and `/ultrareview` commands** added
- **Terminal-matching theme** option
- **Reduced permission prompts**, improved Windows support

### Opus 4.7 Effort Defaults Reversed
After user feedback, Anthropic reversed a prior decision: Opus 4.7 now defaults to `xhigh` effort for all users; all other models default to `high` effort.

### Opus 4.7 Generally Available
Claude Opus 4.7 is GA with improved software engineering, long-horizon coding tasks, and vision upgrades (resolution to 2,576px / ~3.75MP — XBOW visual acuity benchmark jumped from 54.5% to 98.5%).

### Claude Design (Anthropic Labs)
New product for collaborating with Claude to produce visual outputs — designs, prototypes, slides, and one-pagers — directly in-conversation.

### Managed Agents (Platform)
Hosted Claude Platform service for long-horizon agent work, providing stable interfaces for sessions, harnesses, and sandboxes with durable state and safer tool access.

### `ant` CLI
New command-line client for the Claude API: faster API interaction, native Claude Code integration, and YAML-based versioning of API resources.

### Rate Limits API
Admins can now programmatically query rate limits configured for their organization and workspaces.

### Computer Use in CLI (Research Preview)
Claude can open native apps, click through UI, and verify changes from your terminal.

## Announced, Not Yet Released

### Ultraplan (Early Preview)
Draft a plan in the cloud from the CLI, review and comment on it in a web editor, then run it remotely or pull it back local. Still in early preview as of this week.

### Claude Mythos
Announced via Anthropic Red Team (`red.anthropic.com`): a general-purpose language model described as "strikingly capable" at computer security tasks. Limited preview only.

## Community Buzz & Interesting Use Cases

- **GTM automation** is the breakout use case — teams are using Claude Code to build signal-based outbound sequences, adapt messaging by segment, and automate SDR workflows. ~28% of surveyed GTM operators rate it their #1 use.
- **67% of users** say Claude-powered products enabled something previously impossible; 55% say they've already replaced another tool or vendor (most often ChatGPT, an agency, or a contractor).
- **GitHub Routines** are drawing attention from solo developers and small teams who want lightweight CI-adjacent automation without a full pipeline setup.
- **CodePath partnership** — Anthropic is rolling out Claude and Claude Code to 20,000+ students at community colleges, state schools, and HBCUs, with 40%+ from families earning under $50k/year.
- **AWS Bedrock** added Claude Opus 4.7 this week, sparking discussion about enterprise agent workloads running at scale.
